document의 주요 프로퍼티

네이버 홈페이지
이곳은 div 영역입니다.

1. location : 현재문서의 URL 정보를 가진 location
2. URL : 서버의 도메인 이름
3. Title : 문서의 title 제목 문자열
4. head의 id : head 태그의 id 속성 값
5. body color : body 태그의 style 속성에 있는 color 값 / 여기서는 blue
6. domain : 서버의 도메인 이름
7. lastModified : 문서의 마지막 수정일이 적혀있음
8. defaultView : 문서의 기본 뷰를 가진 window 객체
9. 문서의 로드 완료 후 readyState : 문서의 로드 상태를 나타내는 문자열
10. referrer : 현재 문서를 보고 있는 문서의 URL
11. activeElement : 현재 포커스에 적혀있는 요소
12. documentElement의 태그 이름 : 문서의 최상위 요소의 태그 이름
